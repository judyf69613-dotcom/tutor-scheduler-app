資料預設存在目前瀏覽器的 localStorage。換電腦、清除瀏覽器資料，可能會不見，所以建議定期匯出 JSON 備份。

使用方式
  1. 先到「學生資料」新增學生。
  2. 再到「排課 / 上課紀錄」選學生、填日期時間、地點與狀況。
  3. 課後可以把狀態改成「已完成」，並補上理解程度、作業完成度、下次追蹤。
  4. 若時間重疊，系統會提醒可能撞課。
  5. 定期到此頁匯出 JSON，避免資料遺失。
