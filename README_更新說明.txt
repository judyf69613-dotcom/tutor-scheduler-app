家教排課 App v3 更新內容

本版新增：
1. 月排程表：在「排程表」頁籤下方新增月曆式排程，可切換上一月、本月、下一月，也可依學生篩選。
2. 家長課程提醒：可從單筆課程、課程列表或月排程複製提醒文字，格式如下：

再次跟您確認6月課程如下

上課日期：6/20
上課時間：18:30-20:30

3. 家長上課狀況留言：在學生資料的量表下方新增「給家長的上課狀況留言」，勾選量表後可自動產生較委婉的回報文字，並一鍵複製。
4. 月排程 PDF 與本月 .ics：月排程可匯出 PDF，也可匯出本月課程加入行事曆。
5. 保留 v2 資料：本版會自動讀取舊版 v2/v1 儲存在瀏覽器中的資料。

上傳 GitHub Pages 時，請將以下檔案放在 repository 最外層：
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- README_更新說明.txt

更新方式：
1. 解壓縮本 ZIP。
2. 到 GitHub repository 的 Code 頁面。
3. Add file → Upload files。
4. 把上述檔案全部上傳並 Commit。
5. 等 1–5 分鐘後重新整理 GitHub Pages 網址。

如果手機主畫面 App 沒更新：
- 先刪掉舊的主畫面圖示。
- 用 Safari 重新打開 GitHub Pages 網址。
- 分享 → 加入主畫面。
